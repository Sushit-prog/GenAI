STATISTICS NOTES: CONVERGENCE vs COVARIANCE
========================================
File: notes.txt
Purpose: Quick reference for statistical project work
Date: 2026-04-05

----------------------------------------
QUICK SUMMARY
----------------------------------------
- COVARIANCE: Measures how two random variables change TOGETHER at a fixed sample size/distribution.
- CONVERGENCE: Describes how a sequence of random variables, estimators, or distributions BEHAVES AS SAMPLE SIZE GROWS (n → ∞).
- They are fundamentally different: one is a descriptive measure of relationship, the other is an asymptotic/limit property.

----------------------------------------
1. COVARIANCE
----------------------------------------
Definition:
  A measure of the joint variability of two random variables X and Y.
  Indicates whether increases in one variable correspond to increases or decreases in the other.

Formula:
  Population: Cov(X, Y) = E[(X - μ_X)(Y - μ_Y)] = E[XY] - E[X]E[Y]
  Sample:     cov(x, y) = Σ[(x_i - x̄)(y_i - ȳ)] / (n - 1)

Interpretation:
  > 0  : Positive linear association (move in same direction)
  < 0  : Negative linear association (move in opposite directions)
  = 0  : No linear association (does NOT imply independence unless jointly normal)

Properties:
  - Scale-dependent (units = unit(X) * unit(Y))
  - Symmetric: Cov(X, Y) = Cov(Y, X)
  - Bilinear: Cov(aX + bY, Z) = aCov(X, Z) + bCov(Y, Z)
  - Related to variance: Var(X) = Cov(X, X)
  - Standardized version = Pearson correlation coefficient

Common Project Uses:
  - Exploratory data analysis (EDA)
  - Feature selection & multicollinearity checks
  - Portfolio risk & finance modeling
  - PCA & covariance matrix decomposition

----------------------------------------
2. CONVERGENCE (IN STATISTICS/PROBABILITY)
----------------------------------------
Definition:
  Describes the limiting behavior of a sequence of random variables or estimators as n → ∞.
  Answers: "Does my statistic stabilize or approach a target value/distribution?"

Main Types:
  1. Convergence in Probability (→_p)
     - P(|X_n - X| > ε) → 0 as n → ∞
     - Used for consistency of estimators

  2. Almost Sure Convergence (→_a.s.)
     - P(lim_{n→∞} X_n = X) = 1
     - Stronger than convergence in probability

  3. Convergence in Distribution (→_d)
     - CDF of X_n approaches CDF of X at continuity points
     - Basis for Central Limit Theorem

  4. L^p Convergence (e.g., Mean-Square)
     - E[|X_n - X|^p] → 0
     - Implies convergence in probability

Key Theorems:
  - Law of Large Numbers (LLN): Sample mean converges to population mean
  - Central Limit Theorem (CLT): √n(X̄_n - μ) converges in distribution to N(0, σ²)

Common Project Uses:
  - Proving estimator consistency
  - Justifying asymptotic confidence intervals & hypothesis tests
  - Simulation studies (checking stability with larger n)
  - ML: Theoretical guarantees for learning algorithms

----------------------------------------
3. KEY DIFFERENCES
----------------------------------------
| Aspect          | Covariance                          | Convergence                          |
|-----------------|-------------------------------------|--------------------------------------|
| Nature          | Descriptive measure                 | Asymptotic/limit property            |
| Applies to      | Two random variables (fixed n)      | Sequence of RVs/estimators (n → ∞)   |
| Question Answered| "Do X and Y move together?"        | "Does my estimator stabilize as n grows?" |
| Units           | Product of units of X and Y         | Unitless (conceptual limit behavior) |
| Mathematical    | Expectation-based formula           | Limit-based (probability/distribution)|
| Project Context | EDA, correlation, multivariate models| Consistency proofs, asymptotics, theory |
| Common Confusion| Often confused with correlation     | Often confused with optimization convergence |

----------------------------------------
4. PRACTICAL NOTES FOR YOUR PROJECT
----------------------------------------
- If analyzing variable relationships → use COVARIANCE (or correlation).
- If studying estimator behavior, sample size effects, or theoretical guarantees → use CONVERGENCE.
- They CAN intersect: The sample covariance matrix converges (in probability) to the true population covariance matrix as n → ∞ (by LLN).
- In code (Python/R):
    * Covariance: numpy.cov(), pandas.DataFrame.cov(), R::cov()
    * Convergence: Usually checked analytically or via simulation plots (trace plots, error vs n curves, coverage rates)
- WARNING: "Convergence" in ML often refers to optimization (loss decreasing), which is different from statistical convergence of estimators. Clarify your context!
- Zero covariance ≠ independence (except for multivariate normal).
- Convergence in distribution does NOT imply convergence in probability.
